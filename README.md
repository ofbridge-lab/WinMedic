# WinMedic
Outil d'automatisation intelligent pour la réparation Windows. Il orchestre les commandes CHKDSK, DISM et SFC dans l'ordre logique pour corriger corruptions, lenteurs au démarrage et erreurs de mise à jour. Supporte la réparation en ligne ou via ISO. Une alternative automatisée à DISM GUI pour un système sain en un clic.
